# Notion Automations!

Webhook handlers for various automations in our Notion workspace!

## Setup

Install dependencies:

```bash
bun install
```

Set environment variables:

```bash
cp .env.example .env
# Fill in your secrets
```

## Development

```bash
bun dev
```

Built with [Hono](https://hono.dev/) and [Bun](https://bun.sh/) :)
